Why this page exists

Demo intent

This route gives us a clean launch point for the project. It demonstrates the installed baseline experience without requiring ActiveTags, and it keeps the example grounded in the standalone artifact we plan to ship.

Persistent chrome

The masthead and ticker remain outside the swap target so they stay stable while the main content changes.

Useful contrast

The Offroad links still trigger full page loads, which makes it easy to compare SPA versus non-SPA behavior.

Cleaner verification

A richer layout makes it much easier to spot partial swaps, title updates, and missed popstate restores.
