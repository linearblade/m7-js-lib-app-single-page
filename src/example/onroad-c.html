Edge branch route

Strong popstate checkpoint

Ending a branch here and then stepping back through the browser helps validate that popstate restoration is driven by the saved route state rather than by assumptions about what came before.

Expected behavior
  • Back once should return to Onroad B.
  • Back again should return to Onroad A.
  • The shell, nav, and ticker should stay stable while only the route body changes.
Exactness matters

A good popstate restore feels like a precise replay, not a close approximation.

Branch symmetry

The A/B/C family mirrors the numbered pages but with a different route shape.

Easy to debug
